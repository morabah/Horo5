# **Section Update 1: Pricing Architecture**

**\[UPDATED\]**  Replaces v1.1 “Attainable premium / middle premium, reviewed periodically.”

## **New Pricing Position Statement**

HORO occupies the **premium-mid band (799–899 EGP)** in the Egyptian graphic tee market. This positions the brand above mass-market and marketplace prints (119–400 EGP), at parity with premium streetwear (760–950 EGP), and below the ultra-premium wearable-art tier (1,300–1,550 EGP). The price earns its position through original artist illustration, curated meaning, quality DTF execution, and thoughtful packaging.

## **Tiered Price Architecture**

| Tier | Price Band (EGP) | Purpose | When to Use |
| :---- | :---- | :---- | :---- |
| Core | 799 | Primary anchor; covers artist pay \+ margin | Standard collection releases |
| Premium Drop | 899–999 | Limited artist collabs, exclusive themes | Seasonal drops, artist features |
| Gift Bundle | 999–1,199 | Tee \+ story card \+ gift wrap | Gifting occasions: Eid, birthdays, graduation |
| Intro / Test | 649 | Entry price for new customer acquisition | First-time buyer offers, A/B testing only |

**Rule:** 649 EGP is a test and acquisition price, not the brand’s anchor. All brand communication, product pages, and advertising default to the 799 EGP tier unless a specific promotional test is running. Never discount below 649\.

# **Section Update 2: Bilingual Language Strategy**

**\[UPDATED\]**  Replaces v1.1 English-only mandate.

The market research confirms that Egyptian social commerce operates predominantly in Arabic, with 98.4% of online purchasers preferring cash payment through Arabic-language channels (Facebook, WhatsApp). HORO adopts a **split-language model**: English for brand identity, Arabic-led for commerce.

| Touchpoint | Language | Rationale |
| :---- | :---- | :---- |
| Brand name, tagline, logo | English always | Premium positioning; global readability |
| Product names & themes | English primary | Maintains brand vocabulary metaphor |
| Packaging & tags | English primary | Quality cue; unboxing experience |
| Website UI & navigation | English primary, Arabic option | Bilingual toggle available |
| Social media captions | Bilingual (Arabic-led) | Matches how audience discovers and engages |
| Instagram/TikTok ads | Arabic-first | Conversion optimized for reach |
| DMs & WhatsApp service | Arabic-first | Customers ask in Arabic; respond naturally |
| Product descriptions | Bilingual (English \+ Arabic) | Trust-building; removes language barrier |

**Rule:** The brand’s voice character traits (perceptive, warm, culturally grounded) apply equally in both languages. Arabic copy is not a translation of English—it is native Arabic that carries the same personality.

# **Section Update 3: Glassmorphism Visual System**

**\[NEW\]**  Extends v1.1 Section 3 (Visual Identity) with a modern interface design layer.

## **Why Glassmorphism for HORO**

Glassmorphism—the frosted-glass UI style using transparency, background blur, subtle shadows, and layered depth—is among the most influential visual trends in 2025–2026 web and app design. For HORO, it serves three strategic purposes:

* **Visual hierarchy for art:** Frosted glass panels let product illustrations remain the hero while UI elements (pricing, size selectors, CTAs) float on translucent overlays—drawing attention to the art without competing with it.

* **Premium perception:** Glass effects signal sophistication and modernity. Research on color psychology confirms that design sophistication directly influences perceived product quality and willingness to pay premium prices.

* **Warm depth over cold flatness:** Combined with HORO’s warm Papyrus \+ Obsidian palette, glassmorphism creates inviting layers rather than sterile flat UI, reinforcing the “perceptive friend” brand personality.

## **Updated Color Palette: Core \+ Glass Frost Layer**

### **Core Palette (unchanged from v1.1)**

| Name | HEX | Use |
| :---- | :---- | :---- |
| Obsidian | \#1A1A1A | Primary text, dark backgrounds, product backdrops |
| Papyrus | \#F5F0E8 | Website backgrounds, product cards, packaging |
| Clean White | \#FFFFFF | Text on dark, product photography |
| Warm Charcoal | \#2C2A26 | Secondary text, footer, dividers |

### **NEW: Glass Frost Layer (for glassmorphic elements)**

These semi-transparent tints form the frosted-glass overlays. They are always used at reduced opacity (15–35%) with backdrop-blur.

| Name | HEX | Opacity Range | Use |
| :---- | :---- | :---- | :---- |
| Glass White | \#F8F6F2 | 20–35% | Default product card overlay, navigation bar, modals |
| Frost Blue | \#E3EFF5 | 15–25% | Trust signal badges, info panels, size guide overlays |
| Soft Violet | \#EDE5F5 | 15–25% | Mood theme overlays, “inner world” content cards |
| Warm Glow | \#FFF5E6 | 20–30% | Gift collection cards, CTA hover states, featured badges |
| Mint Frost | \#E6F5EF | 15–20% | Success states, delivery confirmation, “you completed it” moments |

### **Glassmorphism CSS Specifications**

All glassmorphic components follow these parameters:

| Property | Value | Notes |
| :---- | :---- | :---- |
| Background | rgba(248,246,242, 0.25) | Adjust per frost color; never exceed 0.40 opacity |
| Backdrop-filter | blur(16px) | Range: 12–24px. Higher \= more frosted, less background visible |
| Border | 1px solid rgba(255,255,255, 0.18) | Subtle white edge to define the “glass” boundary |
| Border-radius | 16px | Rounded corners reinforce warmth; never use sharp corners |
| Box-shadow | 0 8px 32px rgba(26,26,26, 0.08) | Soft, spread shadow; creates floating depth |

### **Where to Use and Where Not to Use Glassmorphism**

| Use Here | Never Here |
| :---- | :---- |
| Product cards on collection pages | Product photography backgrounds (keep clean) |
| Navigation bar (fixed, scrolling) | Body text areas (readability risk) |
| Size selector overlays | Over detailed illustrations (obscures the art) |
| Add-to-cart modal / quick view | Checkout form fields (needs maximum clarity) |
| Theme category badges | Full-page backgrounds (overwhelming) |
| Featured design spotlight | Mobile text-heavy screens |

**Accessibility rule:** All text on glassmorphic surfaces must meet WCAG AA contrast (4.5:1 for body text, 3:1 for large text). Test with both light and dark mode. If contrast fails, increase the frost opacity or add a solid text-shadow backing.

# **Section Update 4: Product Classification & Navigation**

**\[UPDATED\]**  Replaces v1.1 abstract theme categories.

## **The Problem with Abstract Categories**

The v1.1 guidelines organized products by themes like “mood, personality, culture, life stage, and abstract emotion.” Research on purchasing decisions reveals a critical issue: when external product categorization does not match the buyer’s internal mental model, purchase intention drops significantly. The greater the congruence between how customers naturally think about products and how they are organized, the higher the conversion.

Categories like “abstract emotion” or “life stage” may feel meaningful to the brand team but can confuse shoppers who think in more concrete, occasion-based, or visual terms. Choice overload research further warns that unfamiliar categorization increases decision difficulty and reduces purchase confidence.

## **New Classification System: 3-Axis Navigation**

The revised system uses three intuitive navigation axes that let shoppers enter the collection from whichever mental model feels natural to them. This follows best-practice information architecture principles: **user-centered taxonomy based on how customers actually think, not how the brand organizes internally.**

### **Primary Axis: “Shop by Vibe” (Emotional Entry)**

Replaces abstract categories with immediately recognizable visual-emotional cues. Each vibe is paired with a hero image and one sentence.

| Vibe | What It Replaces | Example Prompt | Visual Cue |
| :---- | :---- | :---- | :---- |
| Bold & Loud | Abstract emotion | “For when you want to be noticed” | High-contrast illustrations, energetic poses |
| Soft & Thoughtful | Mood themes | “For the quiet ones who feel deeply” | Muted tones, introspective portraits |
| Proud & Rooted | Culture themes | “Wear where you’re from” | Egyptian motifs, heritage symbols |
| Weird & Wonderful | Personality themes | “For the ones who don’t fit one box” | Surreal, playful, unexpected art |
| Cosmic | Zodiac/personality | “Your sign. Your style.” | Celestial illustrations, gold accents |

### **Secondary Axis: “Shop by Occasion” (Functional Entry)**

Addresses the gift buyer and event-driven buyer directly.

| Occasion | Target Persona | Key Designs |
| :---- | :---- | :---- |
| Gift Something Real | Gift buyer | Curated gift-ready designs with bundle option |
| Graduation Season | Career pride buyer | Achievement \+ ambition themes |
| Eid & Ramadan | Gift \+ cultural buyer | Seasonal capsule with warm, festive art |
| Birthday Pick | Gift buyer | Personality-matched collections |
| Just Because | Self-expression buyer | Everyday self-treat positioning |

### **Tertiary Axis: “Browse by Artist” (Discovery Entry)**

For the art-lover persona. Each artist gets a profile card with portfolio, style description, and available designs. This axis is secondary—never the homepage default—but available for users who want to explore by creator.

## **Navigation Rules**

* **Homepage default:** Open on “Shop by Vibe” with large hero visuals. The first 5 seconds must show art, not menus.

* **Maximum 5 categories visible:** Never show more than 5 options at any single navigation level. This prevents choice overload.

* **Cross-tagging:** Every design is tagged across all three axes. A single design can appear under “Bold & Loud,” “Gift Something Real,” and its artist page simultaneously.

* **Search bar:** Prominent on every page. Support both English and Arabic search terms.

# **Section Update 5: Visual-First Website Design**

**\[NEW\]**  New section based on UI/UX best practices and fashion communication research.

## **Design Philosophy: Show, Don’t Tell**

Fashion communication research confirms that visual narrative persuasion and emotional engagement are the strongest drivers of brand connection. The HORO website must feel like walking through a gallery, not scrolling through a catalog. Every page follows a visual-first principle: the illustration is the hero; text supports but never leads.

## **Homepage Structure: Scroll Storytelling**

The homepage uses a scroll-based narrative that follows the 7-Element Message structure from storytelling best practices: message, dramaturgy, emotion, voice, soundtrack (ambient motion), economy (concise), and rhythm.

| Scroll Section | Content | Design Treatment |
| :---- | :---- | :---- |
| Hero (0–100vh) | Full-bleed animation or video: art being created, then appearing on a tee. Tagline: “Wear What You Mean.” | Glassmorphic CTA button floating over illustration. Micro-animation on scroll. |
| The Feeling (100–200vh) | “You know that feeling when nothing in your closet says what you’re thinking?” | Parallax text over rotating design previews. Glassmorphic text card. |
| The Vibes (200–300vh) | 5 vibe categories as large, immersive visual cards | Glassmorphic cards with hover reveal. Each card fills 60% of viewport. |
| The Proof (300–400vh) | Close-up print quality shots, fabric detail, customer photos | Split-screen: macro photography left, trust badges right. Glassmorphic info overlay. |
| The Stories (400–500vh) | 2–3 customer stories: “Why I chose this design” | Full-bleed customer photos with glassmorphic quote cards. |
| The Drop (500–600vh) | Latest collection showcase with artist credit | Animated card grid with staggered reveal. |
| The Invite (600vh+) | “Find your word.” CTA to browse collection | Glassmorphic CTA panel with warm glow background. |

## **Product Page: The Conversion Canvas**

The product page is where “Recognition” (Stage 3\) and “Epiphany” (Stage 4\) happen. Research shows that in COD-heavy markets, trust signals matter more than storytelling at the point of purchase.

### **Required Elements (in order of visual priority)**

1. **Hero image gallery:** Minimum 5 images per product: flat lay, on-body front, on-body lifestyle, print close-up detail, size reference shot with model measurements.

2. **Glassmorphic quick-view panel:** Floating overlay showing price, size selector, “Add to Cart” CTA. Stays visible while scrolling through images.

3. **Trust strip:** Horizontal bar below gallery: “220 GSM cotton” \+ “Artist: \[name\]” \+ “Free exchange within 14 days” \+ “COD available.”

4. **Design story card:** Glassmorphic card with the illustration’s meaning: “For the one who...” \+ artist micro-bio (3 lines max).

5. **Size guide overlay:** Glassmorphic modal with visual size chart, model measurements, and “Which size am I?” interactive tool.

6. **Social proof:** Customer photos wearing the design \+ Instagram embed.

## **Mobile-First Imperatives**

The research confirms mobile-first, social-led discovery. Every design decision optimizes for mobile viewport first, desktop second.

* Touch targets: minimum 48px for all interactive elements.

* Image loading: lazy-load with blurred placeholder (progressive JPEG). Product gallery as swipeable horizontal carousel.

* Glassmorphism on mobile: reduce blur to 8–12px for performance. Increase frost opacity to 30–40% for readability.

* Sticky CTA: “Add to Cart” button fixed at bottom of screen on product pages, glassmorphic with Ember (\#E8593C) accent.

# **Section Update 6: Storytelling Design Framework**

**\[NEW\]**  Integrates narrative best practices from fashion communication and digital marketing storytelling research.

## **The 7-Element Content Framework**

Every piece of HORO content—from an Instagram reel to a product page to an email—follows a simplified version of the 7-Element Message structure. This ensures consistency, emotional resonance, and strategic purpose.

| Element | What It Does | HORO Application |
| :---- | :---- | :---- |
| Message | The single thing the audience should think, feel, or do | Every post has one message. “This design says what you can’t.” Not three messages. |
| Dramaturgy | Beginning → middle → end structure | Even a 15-second reel has setup (“you know that feeling”), reveal (the design), and CTA. |
| Emotion | The feeling that hooks attention | Lead with recognition or surprise: “That’s me.” Never lead with product specs. |
| Voice | The narrator’s character | The perceptive friend. Warm, understated, culturally grounded. Same in Arabic and English. |
| Visuals | The art direction and imagery | Warm light, Cairo locations, illustration as hero, glassmorphic overlays for text. |
| Economy | Keep it concise | Social: max 3 sentences. Product description: max 5 sentences. Stories: max 15 seconds. |
| Rhythm | Pacing that maintains interest | Alternate between emotional content and proof content. Never post 3 emotion pieces in a row without a trust/proof piece. |

## **Two Storytelling Strategies (Denotative \+ Connotative)**

Fashion communication research distinguishes between denotative (literal, brand-centered) storytelling and connotative (metaphoric, customer-centered) storytelling. HORO uses both:

* **Denotative (30% of content):** Stories about the brand itself—artist process, print technique, design origins, behind-the-scenes. These build credibility and trust. Maps to content streams 2 (“How This Came to Life”) and 5 (“What Should We Say Next?”).

* **Connotative (70% of content):** Stories about the customer’s world—feelings, identities, occasions, relationships. These build emotional connection. Maps to content streams 1 (“What’s Your Word?”), 3 (“You Completed It”), and 4 (“Give Something That Means Something”).

**Rule:** Never let denotative content exceed 30% of weekly output. The customer is always the hero—not the brand, not the artist, not the founder.

# **Section Update 8: Mandatory Trust Signals**

**\[NEW\]**  Required at every customer touchpoint. Non-negotiable.

In Egypt’s COD-dominant, social-commerce market, trust is the biggest conversion barrier—not awareness. Every touchpoint must include proof, not just story.

| Touchpoint | Required Trust Signals |
| :---- | :---- |
| Product page | 5+ images (incl. close-up print detail \+ model with measurements) / Fabric GSM \+ material / Artist name \+ micro-bio / Exchange policy / COD badge |
| Instagram product post | Close-up detail shot in carousel / “220 GSM cotton” in caption / Artist tag |
| Instagram ad | On-body shot \+ price visible \+ “COD available” |
| WhatsApp reply | Product photo \+ size guide link \+ delivery timeline \+ exchange policy summary |
| Checkout page | Order summary with product image / Expected delivery date / Clear COD vs. prepaid options / Exchange policy link |
| Post-purchase | Order confirmation with tracking link / “Tag us in your first wear” / WhatsApp delivery updates |
| Gift bundle | Gift wrap preview image / Story card sample / Delivery guarantee messaging |
